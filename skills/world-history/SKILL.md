# 🌍 World History — 世界历史查询

回答关于世界历史的查询：事件、人物、朝代、文明、战争、帝国等。

## 触发时机

- 用户询问某个历史事件（"法国大革命是什么时候"）
- 用户询问历史人物（"拿破仑生平"、"成吉思汗"）
- 用户询问某文明或帝国（"古罗马"、"奥斯曼帝国"）
- 用户询问历史背景（"工业革命的影响"）
- 用户提问含"历史上"、"古代"、"什么朝代"、"哪个世纪"

## 回答原则

1. **准确性优先**：年代、人名、地名必须准确，不确定时明确说明
2. **简明扼要**：先给结论，再展开背景，默认 300 字以内
3. **时间轴清晰**：涉及多事件时用时间线排列
4. **中文输出**：统一简体中文，专有名词附原文（括号内）

## 执行流程

### 1. 判断查询类型

| 类型 | 示例 | 处理方式 |
|------|------|----------|
| 时间点 | "一战什么时候开始" | 直接回答年份 + 简要背景 |
| 人物 | "凯撒大帝生平" | 生卒年 → 主要成就 → 历史影响 |
| 事件 | "工业革命" | 时间 → 起因 → 经过 → 影响 |
| 文明/帝国 | "古埃及" | 时间跨度 → 地理 → 主要成就 → 衰亡 |
| 对比 | "唐朝 vs 拜占庭" | 并列时间线 + 各自特点 |

### 2. 总纲定位，搜索补充

- 先读 `references/world-history-overview.md` 的「核心矩阵」与「各地区纵向时间线」，按时代 × 地区定位，从矩阵锚点给出公认史实
- 矩阵只到朝代/文明粒度；冷门人物、具体数字、争议解释用 `WebSearch` 核验
- 搜索关键词：`site:wikipedia.org` 或 `历史 + 关键词`

### 3. 输出格式

```
📅 [事件/人物名]

**时间**：XXXX 年 — XXXX 年
**地点**：…
**概述**：一句话核心描述

**背景**
…

**主要经过 / 成就**
- …
- …

**历史影响**
…
```

> 简单问题（如"哥伦布哪年发现美洲"）直接一行回答，不套模板。

## 核心规则

- 年代有争议时注明"约"或给出范围
- 不混淆相似人名（如"亚历山大大帝"与其他亚历山大）
- 涉及敏感或有争议的历史解释，呈现主流史学观点，注明存在争议
- 不做道德评判，客观陈述史实
