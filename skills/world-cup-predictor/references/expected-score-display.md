# 小组赛预期比分显示方法

用户反馈: 模态比分(mode score)全是1:0/0:0/0:1, 每场比赛看起来都一样。

## 问题

```python
# ❌ 旧方法: 选出现频率最高的单一比分
best_score = max(st["goals"].items(), key=lambda x: x[1])
print(f"{best_score} ({pct:.0f}%)")  # → "1:0 (12%)"
```

足球比赛中1:0是最常见的单一比分(2022 WC占15%)。用模态值显示会让72场比赛看起来都一样。

## 最终方案: Top 3模态比分

用户反馈: 平均比分(2.7-0.6)虽然展示差距但用户想要整数比分。最终方案是**显示出现频率最高的3个比分**。

```python
sorted_scores = sorted(st["goals"].items(), key=lambda x: x[1], reverse=True)
top3 = " ".join(f"{sc}({cnt/N_SIM*100:.0f}%)" for sc, cnt in sorted_scores[:3])
# → "1:0(17%) 2:0(15%) 0:0(12%)"
```

**效果**:
- 展示多可能比分, 避免"全是1:0"观感
- 用户看到概率分布而非单个值
- 强弱差通过Top 3的组合清晰可见

**示例**:
```
法国 vs 塞内加尔  1:0(17%) 2:0(15%) 0:0(12%)  ← 法国占优 (72%胜率)
英格兰 vs 克罗地亚 0:0(20%) 1:0(16%) 0:1(10%)  ← 势均力敌 (44%/32%/24%)
巴西 vs 海地      2:0(13%) 1:0(13%) 3:0(12%)  ← 强弱分明 (78%胜率)
```

## 历史: 尝试过的方案

1. ❌ **模态比分**: 选出现频率最高的单一比分 (全是1:0, 难看)
2. ❌ **预期平均比分**: 计算场均进球(2.7-0.6) (用户想要整数比分)
3. ✅ **Top 3模态比分**: 显示最可能的3个比分及概率 (最终方案)

## 实现细节

在 `match_stats` 字典中加入场均进球追踪:

```python
# 初始化时增加 t1_g, t2_g 字段
match_stats[key] = {
    "t1_w": 0, "d": 0, "t2_w": 0,
    "goals": {},        # 比分频率字典 (保留用于调试)
    "t1_g": 0, "t2_g": 0  # 总进球累加器
}

# 每场比赛后累加
match_stats[key]["t1_g"] += g1
match_stats[key]["t2_g"] += g2

# 输出时计算均值
avg_g1 = st["t1_g"] / N_SIM
avg_g2 = st["t2_g"] / N_SIM
```

## 效果对比

| 方法 | 巴西vs海地 | 英格兰vs克罗地亚 | 法国vs伊拉克 |
|:----|:---------:|:--------------:|:----------:|
| 模态比分 | 1:0 (9%) | 0:0 (20%) | 2:0 (10%) |
| 预期比分 | **2.7-0.6** | **1.2-0.8** | **2.7-0.3** |

预期比分更清晰展示了强弱差距(2.7 vs 0.6)和比赛态势(1.2 vs 0.8是势均力敌)。

## 注意事项

- 预期比分是实数(2.7), 不是实际可能的比分
- 结合胜/平/负概率一起看, 全面评估比赛
- 极端值(如5+)表示强弱差距极大
