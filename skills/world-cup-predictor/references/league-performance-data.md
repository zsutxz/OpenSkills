# 联赛表现评分 (League Performance Score)

## 算法 (v4.9)

三维评分，每队以 `(豪门球员数, 首发率%, 个人状态)` 三元组评估：

```
score = min(40, elite_count × 5) + starter_pct × 0.4 + min(20, form × 4)
```

| 子维度 | 权重 | 说明 |
|:------|:---:|:-----|
| 🏟️ 豪门效力 | 40% | 球员在欧冠级俱乐部的数量 |
| 🔄 首发率 | 40% | 国脚在俱乐部的常规首发比例 |
| ⚡ 个人状态 | 20% | 锋线进球/中场组织/防线近期表现 |

## 评分示例

| 球队 | 三元组 | 总分 | 说明 |
|:----|:-----:|:---:|:-----|
| France | (8, 85, 10) | 94 | 姆巴佩/登贝莱/格列兹曼等8人在豪门 |
| Brazil | (7, 80, 8) | 87 | 维尼修斯/罗德里戈/拉菲尼亚 |
| Argentina | (6, 82, 7) | 83 | 劳塔罗/阿尔瓦雷斯 + 卫冕冠军 |
| England | (7, 78, 5) | 86 | 凯恩/贝林厄姆/萨卡 |
| Norway | (2, 50, 5) | 50 | 哈兰德+厄德高双星，其余队友较弱 |
| Japan | (0, 42, 2) | 25 | 零豪门，部分五大联赛球员 |
| Haiti | (0, 3, 0) | 1 | 基本无五大联赛球员 |

完整数据见 `compute_ratings.py` 的 `compute_league_performance()` 函数。
