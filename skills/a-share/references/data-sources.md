# 数据源：东方财富免费行情接口

本 skill 的全部数据来自东方财富（EastMoney）的**公开行情接口**，无需注册、无需 API key、无需 pip 依赖。返回均为 JSON，经 `scripts/data_sources.py` 的 `fetch_powershell()` 抓取——**urllib 直连优先**（快、约 0.2s/次、无中文乱码），连续失败时回退 PowerShell `System.Net.WebClient` 桥接（慢但可绕过部分网络限制）。

> ⚠️ 这些是**非官方**公开接口，字段编号或 URL 可能随官方调整而变化。本文档记录撰写时的字段含义；若抓取结果异常，优先回到这里核对字段编号。

## 通用约定

### 域名

| 域名 | 用途 |
|------|------|
| `push2.eastmoney.com` | 实时行情快照、排行（clist）、个股实时 |
| `push2his.eastmoney.com` | 历史 K 线、历史资金流 |

### `secid`（标的唯一标识）

格式为 `市场前缀.代码`：

| 代码开头 | 市场 | 前缀 | 示例 |
|----------|------|------|------|
| `6` | 沪市主板 / 科创板 | `1` | `1.600519`（贵州茅台） |
| `0`、`3` | 深市主板 / 创业板 | `0` | `0.000001`（平安银行）、`0.300750`（宁德时代） |
| `8`、`4` | 北交所 | `0` | `0.830799` |

判断逻辑见 `data_sources.py: market_of(code)`。

### `ut`（鉴权 token，公开固定值）

不同接口对 `ut` 容忍度不同，下列两个公开值通用：

- `bd1d9ddb04089700cf9c27f6f7426751`（列表类常用）
- `fa5fd1943c7b386f172d6893dbbd10d0`（K 线类常用）

### `fs`（市场筛选，用于 clist）

沪深 A 股全集：`m:0+t:6,m:0+t:80,m:1+t:2,m:1+t:23,m:0+t:81+s:2048`

含义：深市主板 `t:6`、创业板 `t:80`、沪市主板 `t:2`、科创板 `t:23`、北交所 `t:81+s:2048`。

## 接口清单

### 1. 全市场实时快照 `clist/get`（第 1 层·1 次请求）

```
http://push2.eastmoney.com/api/qt/clist/get
  ?pn=1&pz=5000&po=1&np=1
  &ut=bd1d9ddb04089700cf9c27f6f7426751
  &fltt=2&invt=2&fid=f3
  &fs=m:0+t:6,m:0+t:80,m:1+t:2,m:1+t:23,m:0+t:81+s:2048
  &fields=f2,f3,f4,f5,f6,f7,f8,f9,f12,f14,f15,f16,f17,f18,f20,f21,f23
```

返回 `data.diff` 为对象数组。字段：

| 字段 | 含义 | 单位/说明 |
|------|------|-----------|
| `f2` | 最新价 | 元 |
| `f3` | 涨跌幅 | % |
| `f4` | 涨跌额 | 元 |
| `f5` | 成交量 | 手 |
| `f6` | 成交额 | 元 |
| `f7` | 振幅 | % |
| `f8` | 换手率 | % |
| `f9` | 动态市盈率 | 倍（亏损为负） |
| `f12` | 代码 | 如 `600519` |
| `f14` | 名称 | 如 `贵州茅台` |
| `f15` / `f16` | 最高 / 最低 | 元 |
| `f17` / `f18` | 今开 / 昨收 | 元 |
| `f20` | 总市值 | 元 |
| `f21` | 流通市值 | 元 |
| `f23` | 市净率 | 倍 |

> 说明：`fid=f3` 按涨跌幅降序；`fltt=2` 价格保留两位小数。一次可拿全市场 ~5000 只。

### 2. 个股日 K 线 `kline/get`（第 3 层·逐只）

```
http://push2his.eastmoney.com/api/qt/stock/kline/get
  ?secid={market}.{code}
  &ut=fa5fd1943c7b386f172d6893dbbd10d0
  &fields1=f1,f2,f3,f4,f5,f6
  &fields2=f51,f52,f53,f54,f55,f56,f57,f58,f59,f60
  &klt=101&fqt=1&end=20500101&lmt=60
```

- `klt=101` 日线；`fqt=1` 前复权；`lmt=60` 取近 60 根。
- `data.klines` 为字符串数组，每根格式：
  `日期,开,收,高,低,成交量,成交额,振幅,涨跌幅,涨跌额,换手率`
- 解析见 `fetch_stock_kline()`。

### 3. 指数 K 线 / 实时

K 线同上，secid 用指数代码：上证 `1.000001`、深成 `0.399001`、创业板 `0.399006`、科创50 `1.000688`。

实时快照 `stock/get`：
```
http://push2.eastmoney.com/api/qt/stock/get?secid={secid}
  &ut=bd1d9ddb04089700cf9c27f6f7426751
  &fields=f43,f44,f45,f46,f47,f48,f57,f58,f60,f169,f170
```
`f43` 最新价、`f170` 涨跌幅、`f48` 成交额、`f58` 名称、`f60` 昨收。

### 4. 主力资金流排行 `clist/get`（资金面）

```
http://push2.eastmoney.com/api/qt/clist/get
  ?fid=f62&po=1&pz=200&pn=1&np=1
  &ut=bd1d9ddb04089700cf9c27f6f7426751
  &fltt=2&invt=2
  &fs=m:0+t:6,m:0+t:80,m:1+t:2,m:1+t:23
  &fields=f12,f14,f62,f184,f66,f69,f72,f75,f78
```

| 字段 | 含义 |
|------|------|
| `f62` | 主力净流入额（元） |
| `f184` | 主力净流入占比（%） |
| `f66` / `f69` | 超大单净流入额 / 占比 |
| `f72` / `f75` | 大单净流入额 / 占比 |
| `f78` | 小单净流入额 |

### 5. 行业板块 `clist/get`（板块热度）

```
http://push2.eastmoney.com/api/qt/clist/get
  ?pn=1&pz=50&po=1&np=1
  &ut=bd1d9ddb04089700cf9c27f6f7426751
  &fltt=2&invt=2&fid=f3&fs=m:90+t:2
  &fields=f2,f3,f4,f12,f14,f104,f105,f128,f136
```

`fs=m:90+t:2` 为行业板块。`f14` 板块名、`f3` 涨跌幅、`f104/f105` 上涨/下跌家数、`f128` 领涨股。

## 关于北向资金（重要约束）

自 **2024-08-19** 起，沪深港通**不再实时披露北向资金成交额、每日额度**（仅盘后披露持股余额汇总）。因此：

- 本 skill **不使用北向资金实时因子**；
- 资金面以**主力净流入（`f62`）+ 大单/超大单**为代理；
- 历史北向持股仅可作长线参考，不纳入日内打分。

## 降级策略

`fetch_powershell()` 在网络异常、超时、HTTP 错误时返回 `None`。各 `fetch_*` 函数在 `None` 或 JSON 解析失败时返回空集合（`[]` / `{}`），**绝不抛异常中断**。缺失因子的股票评分时该因子置中性分，保证报告仍能生成。

**限流**：东方财富对短时间高频请求会限流（返回 502）。`fetch_market_snapshot` 翻页带 0.08s 节流、失败重试、连续 3 页失败才停止。若整批失败（明显被限流），间隔几分钟后再跑即可恢复。
